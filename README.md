Craftlet: Survival Edition

A lightweight browser-based survival game focused on resource gathering, crafting, exploration, and combat.

Overview

Craftlet: Survival Edition places players in a pixel-inspired world where survival depends on how efficiently resources are gathered, tools are crafted, and enemies are defeated.

Starting with basic equipment, players must collect materials, upgrade their gear, and explore the environment while facing increasingly challenging threats. The game is designed to provide a simple yet engaging survival experience that can be played directly from a web browser.

Features

- Resource gathering system
- Crafting and progression mechanics
- Multiple tools and equipment upgrades
- Survival-focused combat
- Enemy encounters and exploration
- Lightweight browser gameplay
- Desktop and mobile support
- No installation required

Gameplay

The objective is simple: survive and progress.

Players begin with limited resources and must:

1. Gather materials from the environment.
2. Craft tools and equipment.
3. Defeat hostile creatures.
4. Upgrade their inventory and capabilities.
5. Explore the world and survive for as long as possible.

Every resource collected contributes to progression, making strategic decisions an important part of the experience.

Controls

Desktop

Key| Action
WASD| Move
SPACE| Attack
1-9| Select Tool
Q| Craft

Mobile

- Virtual Joystick → Move
- Attack Button → Combat
- Inventory Button → Inventory Access
- Craft Button → Crafting
